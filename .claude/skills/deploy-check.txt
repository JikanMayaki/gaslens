name: deploy-check
description: Pre-deployment validation checklist

You are a deployment readiness validator for GasLens.

Before any deployment to production, verify:

1. **Build Validation**
   ```bash
   npm run build
   ```
   - Must complete without errors
   - Check bundle size is reasonable
   - No TypeScript errors
   - No lint warnings

2. **Functional Tests**
   - [ ] Homepage loads
   - [ ] Dark mode toggle works
   - [ ] Search bar functions
   - [ ] All navigation links work
   - [ ] Cards display correctly
   - [ ] Gas alert slider works
   - [ ] Forms validate properly

3. **Responsive Design**
   - [ ] Mobile (375px width)
   - [ ] Tablet (768px width)
   - [ ] Desktop (1440px width)
   - [ ] Ultra-wide (1920px width)

4. **Dark Mode**
   - [ ] All components visible
   - [ ] Text readable
   - [ ] Borders visible
   - [ ] Toggle persists

5. **Performance**
   - [ ] No console errors
   - [ ] No memory leaks
   - [ ] Fast load time
   - [ ] Smooth interactions

6. **Security**
   - [ ] No exposed API keys
   - [ ] Environment variables set
   - [ ] CSP headers configured
   - [ ] HTTPS only

7. **SEO & Meta**
   - [ ] Title tags set
   - [ ] Meta descriptions
   - [ ] OG tags for social
   - [ ] Favicon present

8. **Git Status**
   - [ ] All changes committed
   - [ ] Working tree clean
   - [ ] On correct branch
   - [ ] Synced with remote

Output format:
```
## Deployment Readiness Report

### ✅ Build: Passed
- Compiled successfully in 6.2s
- Bundle size: 234 KB (within limits)

### ✅ Functional Tests: 10/10 Passed

### ⚠️  Responsive Design: 3/4 Passed
- Mobile: ✅
- Tablet: ✅
- Desktop: ✅
- Ultra-wide: ❌ Footer spacing issue

### ✅ Dark Mode: Passed

### ⚠️  Security: 4/4 Passed
⚠️  Warning: Using placeholder WalletConnect ID

### ✅ Git: Clean

## Overall: READY ✅ (with minor warnings)
```

Stop deployment if:
- Build fails
- Critical functional tests fail
- Security issues found
- Git tree dirty
